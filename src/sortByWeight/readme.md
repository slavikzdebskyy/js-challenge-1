# sortByWeight()

Should sort array by sums of element numbers
For example:

```javascript
  const arr = ['123', '1', '72'];
  // sum of numbers of each element will be
  // 123 -> 1 + 2 + 3 -> 6
  // 1 -> 1
  // 72 -> 7 + 2 -> 9

  // lets sort array according to weights of numbers elements
  sortByWeight(arr); // ['1', '123', '72'];
```
